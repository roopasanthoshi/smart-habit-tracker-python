# Smart Habit Tracker with Reward System

A Python-based habit tracking system that motivates users using reward points and consistency tracking.

## Features
- Multiple habit tracking
- Daily progress monitoring
- Reward points system
- Level-up achievements
- Final performance summary
- User interaction using loops and conditions

## Tech Stack
- Python

## How to Run

```bash
python main.py
```
